# Requirements

- Starship

``` shell
curl -sS https://starship.rs/install.sh | sh
```

- Neovim

``` shell
sudo dnf install neovim
```

- swaybg

``` shell
sudo dnf install swaybg
```

- Mise

``` shell
curl https://mise.run | sh
```

- Zoxide

``` shell
mise install zoxide
```

- Node

``` shell
mise install node@lts
mise shell node@lts
```

- Golang

``` shell
mise install go
mise shell go
```
