# Requirements

- Starship

``` shell
curl -sS https://starship.rs/install.sh | sh
```

- Neovim

``` shell
sudo dnf install neovim
```

- swaybg

``` shell
sudo dnf install swaybg
```

- Mise

``` shell
curl https://mise.run | sh
```

- Zoxide

``` shell
mise install zoxide
```

- Node

``` shell
mise install node@lts
mise shell node@lts
```

- Golang

``` shell
mise install go
mise shell go
```

- KDiff3

``` shell
sudo dnf install kdiff3
```

- CopyQ

``` shell
sudo dnf install copyq
```

- Bitwarden

``` shell
flatpak install com.bitwarden.desktop
```

- fd

``` shell
sudo dnf install fd
```

- luarocks

``` shell
sudo dnf install luarocks
```

- Font

``` shell
./font_install.sh
```

- Links

``` shell
./links.sh
```
